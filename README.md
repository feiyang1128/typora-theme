## ✨ 主题使用

1、下载右侧的 `Releases` 最新压缩包。

2、解压后的 `SeeYue 文件夹` 和 `see-yue-xxxx.css 文件`**直接**放入 Typora 的 `themes` 文件夹。

3、重启 Typora，主题选择 `See Yue xxxx` ，开始愉快地写作！
